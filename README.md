# Repositorio de Programación Web 

## Objetivo General
Integrar y documentar los ejercicios prácticos realizados durante la Actividad 1 del Tema 2 de Programación Web. Este repositorio sirve como portafolio y evidencia del aprendizaje progresivo en tecnologías frontend (HTML5, CSS3 y JavaScript), abarcando desde la maquetación web hasta la implementación de lógica computacional, culminando con el despliegue del proyecto en un entorno de producción mediante GitHub Pages.


## Actividades de Programación Web

En este espacio se encuentran todos los ejercicios prácticos realizados en clase para aprender a crear páginas web paso a paso. A continuación, dejo una explicación muy sencilla de lo que hace cada archivo:

* **Ejercicio 1 (Hola Mundo):** Estructura básica de un documento HTML5 utilizando encabezados (`<h1>`) y párrafos (`<p>`).
* **Ejercicio 2 (Formato de Texto):** Práctica con etiquetas de formato de texto para aplicar negritas (`<strong>`), cursivas (`<em>`) y saltos de línea (`<br>`).
* **Ejercicio 3 (Enlaces):** Implementación de hipervínculos externos e internos utilizando la etiqueta `<a>` y controlando la apertura de pestañas con el atributo `target`.
* **Ejercicio 4 (Imágenes):** Inserción de recursos multimedia locales usando la etiqueta `<img>`, configurando su ruta (`src`), texto alternativo (`alt`) y dimensiones.
* **Ejercicio 5 (Integración):** Creación de un mini blog personal que combina el uso de formatos de texto, imágenes locales y enlaces externos.
* **Ejercicio 6 (Secciones Semánticas):** Estructuración correcta de un sitio web utilizando las etiquetas semánticas de HTML5: `<header>`, `<section>` y `<footer>`.
* **Ejercicio 7 (Listas):** Elaboración de una receta de Tacos al Pastor empleando listas ordenadas (`<ol>`), desordenadas (`<ul>`) y listas anidadas.
* **Ejercicio 8 (Tablas):** Diseño de un horario de clases vinculando un archivo CSS externo. Incluye el uso de cabeceras (`<thead>`) y la combinación de celdas y columnas con `rowspan` y `colspan`.
* **Ejercicio 9 (Formulario de Contacto):** Creación de un formulario estructurado con diferentes tipos de entradas de datos (`text`, `email`, `tel`, `textarea`) y vinculación a estilos CSS externos.
* **Ejercicio 10 (Calculadora de Temperaturas):** Integración de HTML, CSS y JavaScript para crear una calculadora funcional que convierte grados Celsius a Fahrenheit.
* **Ejercicio 11 (Convertidor de Distancias):** Calculadora interactiva que procesa datos de entrada para transformar valores numéricos de Kilómetros a Millas.
* **Ejercicio 12 (Convertidor de Divisas):** Aplicación matemática básica configurada para convertir cantidades monetarias de Pesos Mexicanos (MXN) a Dólares Estadounidenses (USD).
* **Ejercicio 13 (Validación de Voto):** Uso de estructuras condicionales en JavaScript para evaluar la edad de un usuario y determinar si tiene el derecho legal a votar.
* **Ejercicio 14 (Arreglos Matemáticos):** Procesamiento de una lista de números ingresada por el usuario, iterando sobre un arreglo para calcular el valor máximo, el valor mínimo y el promedio general.
* **Ejercicio 15 (Objetos en JS):** Mini sistema para registrar estudiantes y sus calificaciones. Utiliza objetos para almacenar la información de forma estructurada, calcular el promedio grupal y extraer automáticamente a los alumnos con las calificaciones más altas y más bajas.
